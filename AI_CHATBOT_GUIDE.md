# AI Chatbot Guide

## 1. Chạy project
```bash
docker-compose up --build
```

## 2. Seed demo data
```bash
python seed_ai_demo.py
```

## 3. Mở frontend
Vào:

```text
http://localhost:4000
```

## 4. Đăng nhập
- demo@bookstore.vn
- demo123

## 5. Mở chatbot
- Nút **AI tư vấn** ở góc dưới bên phải
- Hoặc mục **AI tư vấn** trên navbar

## 6. Chatbot làm được gì
- Gợi ý sách theo ngân sách
- Gợi ý theo thể loại
- Giải thích coupon, hạng thành viên, giao hàng, đổi trả
- Trả lời tự nhiên, không hiện raw behavior metrics
