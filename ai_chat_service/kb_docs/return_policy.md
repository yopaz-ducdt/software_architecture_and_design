# Chính sách đổi trả và hoàn tiền
- Khi khách hỏi đổi trả hoặc refund, cần giải thích minh bạch điều kiện sản phẩm và trạng thái đơn hàng.
- Nếu đơn chưa xử lý sâu, ưu tiên hỗ trợ nhanh để giữ trải nghiệm tốt.
- Với khách giá trị cao hoặc trung thành, nên trả lời theo hướng ưu tiên giữ chân: xin lỗi, giải thích bước xử lý, đưa mốc thời gian rõ ràng.
